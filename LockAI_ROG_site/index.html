Структура интерфейса
3 страницы · один стиль
1. NaniGatoR. Главная панель, описание системы, режимов, проектов и логики работы. Здесь можно оставить только чистую структуру без лишнего текста и добавлять её по мере развития.
2. Local.AI. Работа с локальным ИИ на ROG: ввод запроса, ответ, режимы отображения. Здесь будет связка с локальным сервером, когда ты его поднимешь.
3. Global.AI. Работа с внешним ИИ через твой backend. Фронт остаётся чистым, ключи и техчасть живут на сервере.
